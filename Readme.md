## ğŸŒŸ Fitur Utama

Simulator ini mencakup 5 konsep inti blockchain:

1.  **Hash Generator (SHA-256):**

    - Memahami bagaimana input teks diubah menjadi string hash unik.
    - Real-time hashing saat mengetik.

2.  **Block Mining:**

    - Simulasi struktur blok (Nomor, Data, Nonce, Timestamp, Hash).
    - Mekanisme **Proof-of-Work** sederhana dengan tombol "Mine".
    - Visualisasi perubahan warna background saat blok valid (hijau).

3.  **Blockchain:**

    - Merangkai beberapa blok menjadi satu rantai.
    - Demonstrasi bagaimana perubahan data di blok lama merusak (invalidate) seluruh rantai di depannya.

4.  **ECC Signature (Elliptic Curve Cryptography):**

    - Simulasi pembuatan Public Key & Private Key.
    - Menandatangani pesan (Signing) dan Verifikasi tanda tangan digital.

5.  **Distributed Consensus:**
    - Simulasi transaksi antar User (A, B, C).
    - Mekanisme konsensus antar node peer-to-peer.
